# 🏋️‍♂️ FITPULSE – Fitness & Wellness Website

## 📌 Project Overview

**FITPULSE** is a modern fitness and wellness web application designed to help users maintain a healthy lifestyle. The platform provides structured workout plans, BMI calculation, categorized exercise sections, progress tracking, and a class timetable.

The goal of this project is to create an interactive and user-friendly fitness website that promotes consistency, motivation, and data-driven fitness improvements.

## ✨ Features

### 🧮 BMI Calculator

* Calculates Body Mass Index based on user input
* Provides health classification (Underweight, Normal, Overweight, etc.)

### 🏋️ Workout Plans

* Structured workout routines for different fitness levels
* Includes strength training, cardio, and flexibility exercises

### 📂 Category-Based Exercises

* Exercises grouped by muscle groups or fitness goals
* Easy navigation for users to find relevant workouts

### 📈 Progress Tracking

* Users can monitor their fitness journey
* Helps maintain consistency and motivation

### 📅 Class Timetable

* Displays scheduled fitness sessions
* Helps users plan their workouts effectively

## 🌐 Deployment

This project is deployed using **GitHub Pages**:

1. Go to repository settings
2. Navigate to **Pages**
3. Select branch: `main`
4. Folder: `/ (root)`
5. Save to generate live link
